Вибір мови програмування та IDE

Для виконання даної лабораторної роботи було використано мову функціонального програмування Scheme та IDE DrRacket. Мова Scheme надає великий набір числових типів і операцій з ними, має неявні типи даних та статичну область видимості. DrRacket дозволяє програмувати на великій кількості мов функціонального програмування та має зручний інтерфейс.

Умова завдання

Розробити процедури для розкладання многочлена по степенях двочлена. Многочлен та двочлен задані списком своїх коефіцієнтів та степенями у вигляді f ( x) = a0 x n + a1 x n −1 + ... + a n −1 x + a n та g(x) = b0 x + b1. Для спрощення можна використовувати схему Горнера. Вивести на екран многочлен, який є результатом n −1 + ... + c1 g ( x) + c0 . Значення розкладання, у вигляді P ( x) = cn g ( x) + cn −1 g ( x) многочленів при заданому користувачем значенні змінної не обчислювати. Результат подати в символьному вигляді.
